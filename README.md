# PlanetExplorersSaveGameParser
This is a small program to help analyze Planet Explorers save game files.

Requirements
-----
GNU make
GCC

Building
-----
```shell
make
```

Usage
-----
Run `pesavegameparse`.
```shell
./pesavegameparse
```
